Site Control

Shortcuts

  • Ctrl+Shift+Y / ⌘+Shift+Y — open popup
  • Ctrl+Shift+J / ⌘+Shift+J — run last CSS selector (highlights matches)
  • Right-click anywhere → "Site Control" submenu

Capabilities

  • Pick any element on the page (Inspector)
  • Click / set-value / fill-form (React & Shadow DOM aware)
  • Run arbitrary JS expressions against the active page
  • Scroll element into view or scroll the window
  • Highlight matching elements temporarily
  • Read localStorage, sessionStorage, cookies (non-httpOnly), page meta

Privacy

This extension runs entirely on your machine. It only talks to the active tab and stores your last selector / JS expression locally via chrome.storage.local.

Install / update

  1. Open chrome://extensions
  2. Toggle Developer mode on (top right)
  3. Click Load unpacked → select the project folder
  4. Pinned: the toolbar icon shows the popup; right-click for the context menu
